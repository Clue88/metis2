Metis v1.0.12: What's New?

Welcome back from midwinter recess! We've added some new features that we hope you'll love – this is a recap of recent changes to the platform.

  • Tables for outstanding assignments and upcoming tests now scroll instead of taking up the whole page.
  • The resources tab! Come here to find commonly-used resources for Stuyvesant students.
  • New colors! We've adopted a lighter theme upon popular demand. However, if you'd like to switch back to the old version, click here or go to your profile settings to change it back.
  • Bug fixes and performance enhancements.

In other news, we recently passed 250 users and 12,000 website views! Thank you so much to all of you who have been using this app and have found this project useful – we really appreciate it..
